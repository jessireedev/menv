# menv
menv is a CLI tool that helps to manage your keys on your .env file.

## Functions
- [x] menv --init
- [x] menv --help
- [x] menv --list
- [x] menv --add [key] [value]
- [x] menv --get [key] 
- [x] menv --edit [key] [value]
- [x] menv --delete [key]
